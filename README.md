# ReadmePad

> Markdown online multi projects editor

[![MIT License][license-img]][license-url]

## Install

```bash
git clone https://github.com/sixertoy/readmepad
cd readmepad
npm install
npm start
# se connecter sur http://localhost:9080/
```

## Gulp's Tasks

- bower [preen, css, js, fonts]
    - css
    - js
    - fonts
- build [css, sass, js, html]
    - css
    - html
    - js
    - sass
- server [main, app]
    - app
    - main
